# Disco on a stick

18 Addressable RGB leds to for a party in your Voron.

## BOM
| Quantity | Value                          | Package |
|----------|--------------------------------|---------|
|       15 | 100nF Capacitor                | 0805    |
|       18 | Addressable RGB LED (WS2812B)  | 5050    |
|        2 | JST-XH (2.54) angled connector | 3pin    |
